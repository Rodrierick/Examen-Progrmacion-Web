Listado etiquetas HTML

  1. Etiquetas iniciales o de raíz

    • Etiqueta: <!DOCTYPE html>

      • Indica al navegador que el documento está basado en el estándar HTML5

    • Etiqueta: <html></html>

      • Representa la raíz de un documento HTML. Todos los demás elementos de la estructura HTML deben ser recogidos dentro de estas etiquetas.

  2. Metadatos del documento

    • Etiqueta: <head></head>

      • Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo. El resto de etiquetas de metadatos, irán recogidas dentro de las etiquetas de apertura y cierre del head. Importante explicar que estos metadatos del documento, es información para el navegador y no contenido que será visible en la página web. A excepción de la etiqueta <title> que veremos a continuación.

    • Etiqueta: <title></title>

      • Etiqueta usada para definir el título de la página web.

    • Etiqueta: <link>

      • Se usa para enlazar recursos externos al documento HTML. El ejemplo más común son las hojas de estilos CSS.

    • Etiqueta: <meta>

      • Etiqueta usada para definir otros metadatos que no se pueden definir con una etiqueta HTML especifica. Por ejemplo para definir el autor del sitio, o la descripción del mismo.

    • Etiqueta: <style></style>

      • Etiquetas usadas para introducir código CSS en línea, es decir, en el propio documento HTML.

  3. Etiquetas de secciones o para estructurar el HTML

    • Etiqueta: <body></body>

      • Al contrario que la etiqueta de metadatos <head>, todo lo que quieras mostrar en la página web debe ir recogido dentro de las etiquetas de apertura y cierre de <body>. Este contenido será el que se muestre en la web.

    • Etiqueta: <nav></nav>

      • Usadas para definir el contenido que será la sección de navegación de la web.

    • Etiqueta: <main></main>

      • Se usa para definir el contenido principal del documento. Solamente puede existir uno por documento.

    • Etiqueta: <section></section>

      • Define una sección del documento

    • Etiqueta: <article></article>

      • Define contenido independiente de la web.

    • Etiqueta: <aside></aside>

      • Dentro de estas etiquetas suele alojarse el contenido adicional de la web. Suele ser contenido relacionado con la web pero de poca importancia.

    • Etiquetas de titulo: <h1> <h2> <h3> <h4> <h5> <h6>

      • Son etiquetas HTML muy importantes, ya que son usadas para jerarquizar el contenido de la web. Las etiquetas se usan para explicar brevemente el contenido que irá a continuación.

    • Etiqueta: <header></header>

      • Se usan para definir la cabecera la página web. Suele contener el logotipo, menú de navegación, etc.

    • Etiqueta: <footer></footer>

      • Usadas para definir el pie de página.

  4. Etiquetas para la agrupación de contenido

    • Etiqueta: <p></p>

      • Etiqueta usada para escribir párrafos de texto.

    • Etiqueta: <hr>

      • Etiqueta utilizada para «romper» entre dos secciones de una web. Usada comúnmente como separador.

    • Etiqueta: <pre></pre>

      • Usada para pegar texto manteniendo el pre formato propio del texto.

    • Etiqueta: <blockquote></blockquote>

      • Se usan para indicar que el contenido es texto citado.

    • Etiquetas: de listas


    • Etiqueta: <ol></ol>

      • Etiquetas para crear una lista ordenada.

    • Etiqueta: <ul></ul>

      • Etiquetas para crear una lista des-ordenada.

    • Etiqueta: <li></li>

      • Etiquetas que recogen el contenido de un elemento de una lista, sea ordenada o no.

    • Etiqueta: <dl></dl>

      • Usada para crear una lista de definiciones.

    • Etiqueta: <dt></dt>

      • Representa un término definido por la siguiente etiqueta <dd>.

    • Etiqueta: <dd></dd>

      • Se usa para definir los términos listados antes que él.

    • Etiqueta: <figure></figure>

      • Indica una figura ilustrada como parte del documento HTML5.

    • Etiqueta: <figcaption></figcaption>

      • Utilizada para definir la leyenda de una figura.

    • Etiqueta: <div></div>

      • Etiqueta común utilizada para crear un contenedor genérico.

  5. Etiquetas semánticas para texto

    • Etiqueta: <a></a>

      • Etiqueta utilizada para crear hiperenlaces en el documento HTML.

    • Etiqueta: <strong></strong>

      • Etiqueta para definir una palabra o conjunto de ellas como importantes. Tiene una fuerte importancia en el SEO de la página.

    • Etiqueta: <small></small>

      • Utilizada para dejar un comentario aparte, del tipo una nota de derechos de autoría, u otros textos que no son esenciales para la comprensión del documento.

    • Etiqueta: <cite></cite>

      • Para indicar el título de una obra

    • Etiqueta: <b></b>

      • Esta etiqueta permitira insertar texto en negrita.

    • Etiqueta: <i></i>

      • Nos permite insertar texto en cursiva.

      Cursiva

    • Etiqueta: <em></em>

      • Sirve para enfatizar un texto, es similar a I.

    • Etiqueta: <sub></sub>

      • Usada para crear un subindice

    • Etiqueta: <sup></sup>

      • Usada para crear un supindice

    • Etiqueta: <mark></mark>

      • Usada para resaltar texto

    • Etiqueta: <span></span>

      • Etiqueta HTML sin ningún significado específico. Se usa conjuntamente con los atributos «class» o «id» para atribuirle ciertas características.

    • Etiqueta: <br>

      • Etiqueta utilizada para crear un salto de línea

  6. Etiquetas para incrustar contenido

    • Etiqueta: <img>

      • Etiqueta para «pintar» una imagen en la página web.

    • Etiqueta: <iframe></iframe>

      • Es una etiqueta que sirve para anidar otro documento HTML dentro del documento principal.

    • Etiqueta: <embed>

      • Usada para integrar una aplicación o contenido interactivo externo que no suele ser HTML.

    • Etiqueta: <object></object>

      • Utilizada llamar a un recurso externo de la web. Este recurso será tratado como una imagen, o un recurso externo para ser procesado por un plugin.

    • Etiqueta: <video></video>

      • Se usa para reproducir video en la página web junto a sus archivos de audio y capciones asociadas.

    • Etiqueta: <audio></audio>

      • Usada para cargar en una web un archivo de audio o stream de audio.

    • Etiqueta: <source>

      • Permite a autores especificar recursos multimedia alternativos para las etiquetas de <video> o <audio></audio>

    • Etiqueta: <svg></svg>

      • Se usa para llamar a una imagen vectorizada.

  7. Etiquetas para la creación de tablas

    • Etiqueta: <table></table>

      • Etiquetas de apertura y cierre de una tabla. El resto de etiquetas de la tabla han de ir siempre recogidas entre estas dos etiquetas.

    • Etiqueta: <caption></caption>

      • Usada para indicar el título de la tabla.

    • Etiqueta: <colgroup></colgroup>

      • Etiqueta utilizada para agrupar dos o más columnas de una tabla.

    • Etiqueta: <tbody></tbody>

      • Usada para describir los datos concretos de una tabla.

    • Etiqueta: <thead></thead>

      • Indica el bloque de filas que describen las etiquetas de las columnas de la tabla.

    • Etiqueta: <tfoot></tfoot>

      • Indica los bloques de filas que describen los resúmenes, o datos totales de una columna de una tabla.

    • Etiqueta: <tr></tr>

      • Se usa para indicar una fila de celdas de una tabla.

    • Etiqueta: <td></td>

      • Usada para definir una celda de una tabla.

    • Etiqueta: <th></th>

      • Etiqueta que se usa para definir el encabezado de una celda.

  8. Etiquetas para la creación de formularios

    • Etiqueta: <form></form>

      • Etiqueta de apertura y cierre de un formulario de una página web. El resto de etiquetas de formulario deben ir siempre recogidas entre estas etiquetas de apertura y cierre de formulario.

    • Etiqueta: <fieldset></fieldset>

      • Etiqueta que representa un conjunto o agrupación de elementos de un formulario. «Pinta» un recuadro alrededor de las etiquetas que estén contenidas dentro del <fieldset>

    • Etiqueta: <legend></legend>

      • Etiqueta ligada a <fieldset>. Indica el título del <fieldset>

    • Etiqueta: <label></label>

      • Se usa para definir el nombre o título de un control del formulario.

    • Etiqueta: <input>

      • Pinta un campo de introducción de datos para el usuario. Es de las principales etiquetas de un formulario.

    • Etiqueta: <button></button>

      • Etiqueta utilizada para representar un botón en el formulario.

    • Etiqueta: <select></select>

      • Input que permite una selección entre un conjunto de opciones.

    • Etiqueta: <option></option>

      • Etiqueta ligada a <select>. Permite añadir diferentes opciones al <select>.

    • Etiqueta: <textarea></textarea>

      • Añade un campo al usuario para que pueda introducir texto en unas líneas máximas de texto que el desarrollador puede definir.

Estilos en CSS
  1. Introducción a CSS

    2. Que es CSS?
    CSS(Cascade Style Sheets) Hojas de estilo en cascada, nos sirve para crear diversos estilos para nuestros elementos en HTML. Pudiendo definir de forma estructurada, la manera en la cual se organizarán estos estilos. CSS, nos permite ahorrarnos mucho trabajo, ya que al crear una plantilla de estilos que puede ser aplicacada rapídamente, evitamos tener que crear un estilo individual para cada elemento.

  2. Como usarlo

    3. Existen 3 formas de utilizar CSS

    • Externa
      • Para utilizar el código css de forma externa podemos utililizar un archivo diferente dedicado exclusivamente al código css.
      Para ello utilizamos la etiqueta
      link rel="stylesheet" href="miestilo.css"
    • Interna
      • También podemos usar el código CSS en nuestra página HTML usando la etiqueta:
      <style>
      body{ color: green; }
      </style>
    • En línea

  3. Sintaxis

    4. La sintaxis de CSS es el conjunto de reglas que definen cómo escribir código en CSS. Estas reglas definen cómo se deben estructurar las reglas de estilo y cómo se deben combinar los selectores, propiedades y valores para producir un diseño coherente en una página web.

    • Cambiar el color del texto
    • Cambiar el tamaño de la fuente
    • Cambiar el fondo

  4. Selectores

    5. Los selectores son la forma en que se apunta a los elementos HTML que se quieren estilizar. Los selectores pueden ser de diferentes tipos, como de etiqueta, de clase, de ID, entre otros.

    • Selectores de etiqueta:
      • Estos selectores apuntan a elementos HTML específicos. Por ejemplo, el selector de etiqueta p apuntaría a todos los elementos <p> en la página.
    • Selectores de clase:
      • Los selectores de clase apuntan a elementos HTML que tienen una clase específica. Por ejemplo, el selector .mi-clase apuntaría a todos los elementos que tienen la clase mi-clase.
    • Selectores de ID:
      • Los selectores de ID apuntan a elementos HTML que tienen un ID específico. Por ejemplo, el selector #mi-id apuntaría al elemento que tiene el ID mi-id.

  5. Comentarios

    6. Existen dos tipos de comentarios en CSS:

    • Comentarios de una línea CSS
      • Los comentarios de una línea se especifican con el símbolo barra seguido de un asterisco. Para cerrar el comentario se hace a la inversa, primero asterisco y finalmente la barra. Se pueden situar arriba de cualquier línea o a un lado.

      /* Este es un selector de agrupación que afecta a h1, h2 y p */
      h1, h2, p {
               text-align: center;
               color: deepskyblue;
      }
    • Comentarios multilínea en CSS
      • Los comentarios multilínea en CSS no cambian, simplemente, inicia el comentario en una línea y ciérralo en otra línea. Se utilizan o bien para una explicación o detalle de varias líneas o para anular ciertas partes de tu hoja de estilos, de modo que no tengas que borrarlas por si las necesitas más adelante.

      /* Este comentario anula todo lo que se encuentra entre su apertura y su cierre.
      h1, h2, p {
               text-align: center;
               color: deepskyblue;
      } */

  6. Colores

    7. Los colores en CSS se pueden indicar de cinco formas diferentes: palabras clave, colores del sistema, RGB hexadecimal, RGB numérico y RGB porcentual. Aunque el método más habitual es el del RGB hexadecimal, a continuación se muestran todas las alternativas que ofrece CSS.

    • Palabras clave
      • CSS define 17 palabras clave para referirse a los colores básicos. Las palabras se corresponden con el nombre en inglés de cada color: aqua, black, blue, fuchsia, gray, green, lime, maroon, navy, olive, orange, purple, red, silver, teal, white, yellow

  7. Fondos

    8. CSS permite establecer de forma simultánea un color y una imagen de fondo. En este caso, la imagen se muestra delante del color, por lo que solamente si la imagen contiene zonas transparentes es posible ver el color de fondo.

    • Como aplicar fondos de color con CSS
      • La manera más sencilla de aplicar un fondo es usar la propiedad background-color, indicando a continuación el RGB del color que deseamos aplicar.

      body {
              background-color: #0f1473;
      }
    • Cómo aplicar fondos de imagen con CSS
      • También podemos aplicar un fondo con una imagen, con el atributo background-image, de la siguiente manera.

      body {
              background-image: url(fondo-degradado.png)
      }
    • Otros atributos para definir el fondo con CSS
      • background-repeat
        • Este atributo es el que tiene la responsabilidad de decidir si realmente queremos que una imagen se muestre como un mosaico, o si queremos que aparezca una única vez.

        body {
                background-image: url(patron.png);
                background-repeat: no-repeat;
        }
      • background-position
        • Este atributo te sirve para definir dónde se va a colocar el fondo, sobre el elemento o sobre la página. Inicialmente la posición será "0 0", lo que quiere decir que la esquina superior izquierda de la imagen se colocará en la esquina superior izquierda del elemento al que se haya aplicado el fondo.

        body {
                 background-image: url(fondo.png);
                 background-position: center;
        }
      • background-size
        • Este atributo resulta muy potente para definir no solo el tamaño del fondo sino también cómo va a cubrir todo el elemento donde lo pongamos. Por ejemplo, una configuración muy útil es "cover" que permite que el fondo cubra todo el espacio disponible.

        body {
                 background-image: url(fondo-css.jpg);
                 background-size: cover;
        }
      • background-attachment
        • Este atributo CSS aplica a la manera que tiene el fondo de engancharse al contenedor, en caso que se pueda hacer desplazamiento sobre él. La verdad es que no se usa demasiado, pero puede servir para hacer algunos efectos interesantes. Sus tres valores posibles son scroll, fixed y local. El valor predeterminado es "scroll", lo que produce que el fondo se mueva cuando el contenedor tiene scroll, pero si queremos que el fondo se quede fijo podemos usar fixed.

        body {
                 background-image: url(fondo-degradado.png);
                 background-attachment: fixed;
        }
      • background-origin
        • Define dónde queremos que se pinte el fondo. De nuevo, no se usa mucho. Generalmente los fondos se pintan en todo el elemento, incluyendo el padding. Pero podríamos definir otros comportamientos como por ejemplo que se pinte solamente en el área del contenido, sin incluir el padding.

        div {
                 background-image: url(patron.png);
                 background-repeat: no-repeat;
                 background-origin: content-box;
                 padding: 40px;
                 border: 3px solid red;
        }
    • Múltiples fondos en un mismo elemento con CSS
      • A partir de CSS 3 es posible aplicar muchos fondos de imagen a un mismo elemento. Para elllo simplemento podemos indicar todas las imágenes de fondo que queramos usar, separadas por comas.

      body {
               background-color: #f5f5f5;
               background-image: url(fondo-degradado.png), url(patron.png);
               background-position: 0, center;
               background-repeat: no-repeat, repeat;
               background-size: cover, auto;
               background-attachment: fixed, scroll;
      }

  8. Bordes
    9. Las propiedades básicas y específicas de los bordes en CSS son las siguientes:
    • border-color
    • border-width
    • border-style
    En primer lugar, border-color establece el color del borde, de la misma forma que lo hicimos en apartados anteriores de colores. En segundo lugar, con border-width podemos establecer la anchura o grosor del borde utilizando tanto palabras clave predefinidas como un tamaño concreto con cualquier tipo de las unidades ya vistas.
  9. Márgenes

    10. En el modelo de cajas, los márgenes (margin) son los espacios exteriores de un elemento. Es decir, el espacio que hay entre el borde de un elemento y su exterior.
    • margin-top
      • Establece un tamaño de margen superior.
    • margin-left
      • Establece un tamaño de margen a la izquierda.
    • margin-right
      • Establece un tamaño de margen a la derecha.
    • margin-bottom
      • Establece un tamaño de margen inferior.
  10. Altura/Ancho
    • Anchura
      • La propiedad CSS que controla la anchura de la caja de elementos se denomina width.
    • Altura
      • La propiedad CSS que controla la altura de los elementos se denomina height.
